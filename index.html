Trabajos recientes

Semillero de investigación. Este proyecto lo llevamos construyendo desde el año 2020, se trata del desarrollo de un sistema que permita registrar los proyectos finales de etapa lectiva de los programas de formación de contenidos digitales del Centro para la Industria de la Comunicación Gráfica, SENA Regional Distrito Capital

Proceso de prototipado portales web. Este proyecto fue espectacular, consistio en trabajar en el proceso de prototipado en fases de diseño de creación de 3 sistemas de información; Portal institucional, Super servicios, Portal intranet, construi toda las fases de diseño desde el scketch, wireframe, mockup, prototipo una enorme responsabilidad que me permitio entender la integración de los componentes backend con los frontend

sui.superservicios.gov.co superservicios.gov.co

Sitio web de Casa de cambios en Unicentro Bogotá. Este proyecto es un sistema de información que actualiza los precios de las monedas en tiemporeal atraves del frontend, usamos herramientas de maquetación y diseño adaptable

Sitio Web de Motores de segunda. Este sitio posee un sistema que permite registrar y almacenar los datos de los posibles clientes, para realizar un sistema embudo que permita el engagement con el cliente

Para lograr estos proyectos trabajamos con equipos de trabajo comprometidos, cada uno de ellos especializados para incorporar y proponer en areas como; Marketing Digital, Modelos de Negocio, Diseño Gráfico, Diseño Web y App, Producción Multimedia, Bases de Datos, 3D, Animación, Edición, Sistemas de Información, Inteligencia Emocional, Inteligencia Artificial.
